# Installation 

AegisCrypt is lightweight and only relies on the widely trusted `cryptography` library.

## Prerequisites
*   Python 3.10 or higher.
*   `pip` (Python package installer).

## Step-by-Step Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/_ranveerpardeshi_/AegisCrypt.git
   cd AegisCrypt
   ```

2. **Install dependencies:**
   ```bash
   pip install cryptography
   ```

3. **Verify Installation:**
   Run the interactive menu to ensure everything is working:
   ```bash
   python aegiscrypt.py
   ```
